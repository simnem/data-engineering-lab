# data-engineering-lab
Playground for small projects to reinforce learning..

A collection of hands-on exercises, experiments, and small projects created while learning and improving Data Engineering skills.

This repository serves as a personal sandbox for practicing concepts, testing tools, and building familiarity with common Data Engineering workflows.

## Purpose

The goal of this repository is to develop practical experience with:

- Data ingestion and extraction
- ETL / ELT pipelines
- APIs and data collection
- SQL development
- Python for data processing
- Data modeling
- Databases
- Cloud and distributed data technologies
- Workflow orchestration
